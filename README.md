#  Banking Program (Python CLI)

A simple command-line banking application written in Python.  
Users can view their balance, deposit money, and withdraw funds safely with input validation.

##  Features
- Show current balance
- Deposit money
- Withdraw money (with insufficient funds protection)
- Input validation for negative values
- Simple and user-friendly CLI interface

##  How It Works
The program runs in a loop and displays a menu:
1. Show Balance  
2. Deposit  
3. Withdraw  
4. Exit  

Users select an option, and the program performs the requested action.
